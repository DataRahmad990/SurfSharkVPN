User Guide: Surfshark VPN

Panduan Pengaturan untuk Windows & macOS

01

Pilih Akses Surfshark

Buka aplikasi utama, klik pada ikon Surfshark dan pilih akses yang diinginkan.

02

Download Aplikasi Desktop

Setelah masuk ke dashboard web, unduh aplikasi Surfshark untuk desktop Windows.

03

Pilih Connect Manually

Install dan buka aplikasi. Pada layar login, pilih opsi Connect manually.

04

Dapatkan Kredensial

Kembali ke dashboard web pilih VPN--> Manual Setup--> Open VPN untuk mendapatkan username dan password untuk setup manual.

05

Pilih Lokasi Server

Pilih tab Locations dan pilih negara atau server tercepat (Fastest server).

06

Salin Hostname / IP

Setelah memilih lokasi, klik ikon salin untuk menyalin Server IP atau Hostname.

07

Masukkan Semua Data

Kembali ke aplikasi desktop, masukkan username, password, dan IP yang telah disalin.

08

Hubungkan & Selesai!

Klik Connect. VPN Anda sekarang sudah aktif dan koneksi Anda aman.

01

Pilih Akses Surfshark

Buka aplikasi utama, klik ikon Surfshark, dan pilih akses yang Anda inginkan.

02

Pilih Browser Extension

Di dashboard web, cari opsi "Download the Chrome extension" dan klik Add extension.

03

Tambahkan ke Browser

Anda akan diarahkan ke Web Store. Klik tombol Tambahkan ke Chrome untuk menginstall.

04

Buka & Lewati Tutorial

Setelah terinstall, buka extension dan klik Exit setup untuk melewati tur pengenalan.

05

Login ke Akun Anda

Langkah selanjutnya adalah login menggunakan akun Surfshark Anda di dalam jendela extension. Masukkan email dan password Anda.

Pilih Negara & Hubungkan

Pilih lokasi server yang Anda inginkan (misal: Indonesia) atau gunakan Quick-connect.

Koneksi Terlindungi

Anda berhasil terhubung ke Surfshark VPN. Selamat berselancar dengan aman!
